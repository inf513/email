AYUDA PARA REALIZAR PETICIONES
Configuracion de peticion COMANDO|TABLA|[data1,data2,...] 
lista de Comandos:
ADD : Comando para registrar una peticion.
CON : Comando para buscar un registro.
LST : Comando para listar los registros.
DEL : Comando para eliminar un registro.
UPD : Comando para actualizar un registro.
 
Peticiones comunes
DEL|TABLA|[1] Elimina el registro con el [ID] en el sistema.
CON|TABLA|[1] Busca un registro con el [ID] en el sistema.
LST|TABLA|[]   Lista los registros del sistema.
 
PETICIONES DE CARGO CMD|CARGO|[idCargo,codigo,nombre]
ADD|CARGO|[0,001,AYUDANTE] Registra un cargo en el sistema.
UPD|CARGO|[1,001,AYUDANTE] Modifica un cargo en el sistema.
 
PETICIONES DE GESTION CMD|GESTION|[idGestion,codigo,fechaInicial,fechaFinal]
ADD|GESTION|[0,19,2019-01-01,2019-12-31] Registra una gestion en el sistema.
UPD|GESTION|[1,19,2019-01-01,2019-12-31] Modifica una gestion en el sistema.
 
PETICIONES DE ORDEN DE TRABAJO CMD|ORDEN|[idOrden,idGestion,codigo,nombre,supervisor,area]

UPD|ORDEN|[1,1,001,SAN IGNACIO,NOMBRE SUPERV,1000] Modifica una orden de trabajo en el sistema.
